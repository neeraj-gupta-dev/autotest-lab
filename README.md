# 🧪 AutoTest Lab

> A full-stack MERN portfolio project designed as a modern **live testing playground** for practicing automated UI testing with Selenium, Playwright, or Cypress.

## 📋 Features

### 🔐 Authentication System
- User Registration with robust validation
- Secure JWT-based Login and Session Management
- Protected routing enforcement

### 🛒 E-Commerce Simulation
- Dynamic product catalog 
- Detailed product viewing experience
- Add-to-cart functionality with real-time counter

### 📝 Testing Playgrounds
- **Authentication Forms** — Validations for login & registration
- **Interactive Elements** — Complex inputs, dropdowns, checkboxes, and radios
- **Native Dialogs** — Native `alert()`, `confirm()`, and `prompt()` triggers
- **Live Search** — Instant filtering with debounced input
- **Mock REST API** — E-Commerce data interactions

### 💡 User Experience
- Optimized loading states and micro-animations
- Graceful empty states and error boundaries
- Responsive design tailored for modern browsers

## 🎓 Academic Use (STQA)

This application was originally conceptualized as a **Software Testing & Quality Assurance (STQA)** target application. It provides a stable, predictable, and fully-featured MERN environment constructed with modern best practices, allowing engineers and students to easily write automation scripts against complete user flows.

**Recommended Automation Tools:** Selenium WebDriver, Playwright, Cypress
